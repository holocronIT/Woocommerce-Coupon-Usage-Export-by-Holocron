== Description ==
This plugin permit to export, in csv forma ,the list of orders where a specific coupon code it was used. 
For every coupon return the list of orders with total amount, order date, client info e order items

== Installation ==
Upload the zip file in the Wordpress plugin admin page or directly upload folder in wp-content/plugins directory
